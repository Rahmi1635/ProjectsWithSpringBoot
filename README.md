# 🚀 ProjectsWithSpringBoot

Bu repository, **Spring Boot** kullanarak geliştirdiğim backend projelerini içermektedir.
Amaç, **ölçeklenebilir ve sürdürülebilir backend sistemleri geliştirme yetkinliğimi göstermek** ve gerçek dünya senaryolarına yönelik çözümler üretmektir.

---

## 🎯 Odak Noktaları

Bu repo kapsamında geliştirdiğim projelerde aşağıdaki mühendislik prensiplerine odaklanıyorum:

* Katmanlı mimari (Controller - Service - Repository)
* RESTful API tasarımı
* DTO (Request / Response) ile veri yönetimi
* SOLID prensiplerine uygun yapı
* Temiz kod (Clean Code) yaklaşımı
* Hata yönetimi ve veri doğrulama

---

## 🛠️ Teknoloji Stack

* Java
* Spring Boot
* Spring Data JPA
* Hibernate
* Maven
* Lombok
